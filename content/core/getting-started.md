# Getting Started

Welcome to the **Docutron** documentation preview. This page is loaded from a Markdown file under `content/core/`.

## How navigation works

The left sidebar is built from `manifest.json`. That file is generated by scanning every `.md` file and reading the [YAML](https://yaml.org/) block inside an HTML comment at the top of the file.

Add a new `.md` file, run the manifest script, refresh — the new page appears in the sidebar without editing HTML.

## Previous and next

Footer links are **optional** and defined in each file’s front-matter (`prev` and `next` as paths under `Documentation/`). Omit either key when there is no sensible link—for example, the last page in a topic does not need a “Next” if the following docs are unrelated.

This page only sets **`next`** (there is no previous page in that sequence).

## Sidebar and small screens

The sidebar has its **own scrollbar** when the list is taller than the panel (it no longer stretches to match the article height). On **narrow viewports**, use the **Menu** button to open the navigation drawer; tap outside or press **Escape** to close it.
