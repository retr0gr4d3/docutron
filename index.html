Docutron

A small, static doc viewer: write Markdown + YAML front matter, generate a manifest, ship a fast docs site anywhere you can host files.

Did you know? Creating a docs site with Docutron is as simple as writing Markdown and YAML, then running a single command to generate a manifest and deploy your site. Spin up a http server and you're good to go.

Homegrown

Docutron was made to be a simple, fast, and easy to use docs site generator for my own website and projects. Built from the ground up with simplicity and speed in mind. Edit this homepage to your liking, or bring your own pre-existing index.html file.
